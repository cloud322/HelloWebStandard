CSS Reset

CSS 를 적용하더라도 모든 부라우저에 동일하게 보이는 것은 아님

각 브라우저마다 기본적으로 적용되어 있는 스타일의 차이떄문인데

이것을 CSS Reset 이라는 특수한 기능으로 모든 브라우저의 기본 스타일 여백 등등을 초기회 시킬수 있음
